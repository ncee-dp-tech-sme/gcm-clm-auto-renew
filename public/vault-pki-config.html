ℹ️ About Vault PKI

This application uses HashiCorp Vault's PKI secrets engine to issue and manage certificates.

  • Certificates are issued directly from Vault without ACME protocol
  • Automatic renewal is supported (checks every 3 minutes)
  • Certificates are valid for 30 days by default
  • Vault connection can be pointed to another existing Vault server
  • When switching away from the default Vault setup, the original config is backed up for restore
  • Vault authentication is handled automatically using the configured token source
